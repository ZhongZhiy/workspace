
## 🛠️ 如何创建和使用虚拟环境？

### 1. 创建虚拟环境

```bash
python -m venv myenv
```

这会在当前目录下创建一个名为 `myenv` 的文件夹，里面包含了一个完整的 Python 运行环境副本。

---

### 2. 激活虚拟环境

#### Linux/macOS：

```bash
source myenv/bin/activate
```

---

### 3. 在虚拟环境中安装包

```bash
pip install cyaron
```

这个包只会安装在当前的虚拟环境中，不会影响系统的 Python。

---

### 4. 退出虚拟环境

```bash
deactivate
```

执行后你会回到全局的 Python 环境中。

---

## 🗃️ 查看已安装的包

在虚拟环境中运行：

```bash
pip list
```

可以看到当前环境中安装的所有包。

---

## 📁 虚拟环境结构简介

当你运行 `python -m venv myenv` 后，会生成如下结构的文件夹：

```
myenv/
├── bin/            # 包含 pip 和 python 命令的可执行文件
├── include/        # 编译扩展时用到的头文件
├── lib/            # 所有安装的第三方模块
└── pyvenv.cfg    # 配置文件
```
