# 反码, 补码, 原码
## 原码
符号位为1为负数, 为0为正数, 这种形式就是原码

## 反码
原码除了符号位以外的所有位取反
$$
N = 
\begin{cases}
N, & N > 0 \\
(2^n - 1) - N, & N < 0
\end{cases}
$$
## 补码
- 不考虑进位的有限位数, 两个数相加为0, 则互为补码, 例如:`1010` 和`0110`, 和为`10000`但是只考虑4位也就是`0000`
- 同时补码=反码+1
- 负数的绝对值就是对所有位取反+1, 包括符号位

$$

N = 
\begin{cases} 
N, &N > 0 \\
2^n-N, &N < 0
\end{cases}
$$

# 十进制代码
使用4位二进制码表示一个十进制码
1. 8421码(BCD)
2. 2421码, 0和9, 8和1, 7和2, 6和3, 5和4互为反码, 对于求取对10的补码方便
3. 5211码
4. 余3码, 从3开始代表0, 到12代表9, 两个数相加为10, 余3码恰好为16, 这样便从高位自动产生进位信号
5. 余3循环码, 格雷码的一部分
6. 格雷码, 相邻两个代码之间只有一位发生变化
