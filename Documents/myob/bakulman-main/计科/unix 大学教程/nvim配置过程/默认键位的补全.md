default'（推荐）：使用类似原生补全的快捷键
```lua
    --   <c-y> 接受当前补全项（[y]es），如果 LSP 支持会自动导入
    --       如果 LSP 返回的是 snippet，也会自动展开
    -- 'super-tab'：按 Tab 接受补全
    -- 'enter'：按 Enter 接受补全
    -- 'none'：不绑定任何快捷
    -- 为什么推荐使用 'default' 预设？
    -- 建议阅读 `:help ins-completion` 来了解 Vim 的插入模式补全机制
    -- 真的，认真建议你读一下这个帮助文档，非常有价值
    -- 所有预设都包含以下通用快捷键：
    -- <tab>/<s-tab>：在 snippet 展开的位置之间切换
    -- <c-space>：打开补全菜单，或已打开时显示文档
    -- <c-n>/<c-p> 或 <up>/<down>：选择下一个/上一个补全项
    -- <c-e>：隐藏补全菜单
    -- <c-k>：切换函数签名提示窗
```
