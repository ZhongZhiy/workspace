### 帮助信息

- 按下 `<C-c>` 可以中止所有正在运行的任务。

- 将光标定位在一个插件上并按下 `<CR>`（回车键），可以查看该插件的详细信息。

- 大多数属性可以通过按下 `<K>` 键来悬停查看，这会打开链接、帮助文件、README 文件以及 Git 提交记录。
  - 如果在任意位置对某个插件按下 `<K>` 键进行悬停：
    - 如果该插件有更新或刚刚被更新，则会打开一个 diff 视图；
    - 否则将打开该插件的网页页面。

- 在提交记录或插件上按下 `<d>` 键可以打开 diff 视图。

- 使用 `<]]>` 和 `<[[>` 在不同插件之间导航。

### 键盘快捷键

- **主页** `<H>`：返回插件列表
- **安装** `<I>`：安装缺失的插件
- **更新** `<U>`：更新插件。这也会更新 lockfile
- **同步** `<S>`：执行安装、清理和更新操作
- **清理** `<X>`：清理不再需要的插件
- **检查更新** `<C>`：检查插件是否有更新并显示日志（git fetch）
- **日志** `<L>`：显示最近的更新记录
- **恢复** `<R>`：将所有插件恢复到 lockfile 中的状态
  - 对单个插件使用时：将其恢复到 lockfile 中的状态，或者恢复到光标所在处指定的提交版本
- **性能分析** `<P>`：显示详细的性能分析
- **调试** `<D>`：显示调试信息
- **帮助** `<?>`：切换显示此帮助页面

---

### 插件相关键盘快捷键

- **安装** `<i>`：安装当前插件
- **更新** `<u>`：更新当前插件。这也会更新 lockfile
- **删除** `<x>`：删除当前插件。⚠️注意：即使该插件本应被保留，此操作也会将其删除！
- **检查更新** `<c>`：检查插件是否有更新并显示日志（git fetch）
- **日志** `<gl>`：显示当前插件的更新日志
- **恢复** `<r>`：将当前插件恢复到 lockfile 中的状态，或者恢复到光标所在处指定的提交版本
- **重建** `<gb>`：重新构建插件
- **自定义键** `<localleader>l`：打开 lazygit 的日志界面
- **自定义键** `<localleader>i`：查看插件详情
