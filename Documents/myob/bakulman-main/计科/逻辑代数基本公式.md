## 常用公式
21. $A + AB = A$ 两个乘积相加的时候, 其中一个是另外一个的因子, 则该项是多余的
22. $A + A'B = A + B$ 如果一项的取反是另外一项的因子, 那么这个因子是多余的
23. $AB + AB' = A$ $\implies A(B+B') = A$
24. $A(A + B) = A$ 变量A和包含A的和相乘的时候, 其结果等于A, 其余项可以消除
25. $AB + A'C + BC = AB + A'C$ 如果两个乘积中分别包含A和A'两个因子, 而且这两个乘积的其余因子组成第三个乘积项时, 则第三个乘积项是多余的, 可以消去
26. $A(AB)' = AB'$ 当A和一个乘积项的**非**相乘的时候, A是乘积项的因子, 那么这个因子可以消除
### 逻辑代数基本定理
#### 代入定理
任何一个包含变量A的逻辑等式中, 如果用另外一个逻辑式代入所有A的位置, 则仍然成立.
