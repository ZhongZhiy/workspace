在 C++ 中，你可以通过多种方式手动给多个 `std::map` 变量赋初值。以下是一些常用方法：

---

### 1. **直接插入键值对**

使用 `map` 的 `insert` 或 `operator[]` 赋初值：

```cpp
#include <iostream>
#include <map>
using namespace std;

int main() {
    map<int, string> map1;
    map<string, int> map2;

    // 使用 [] 赋值
    map1[1] = "One";
    map1[2] = "Two";

    map2["Alice"] = 30;
    map2["Bob"] = 25;

    // 使用 insert 赋值
    map1.insert({3, "Three"});
    map2.insert({"Charlie", 35});

    // 输出验证
    for (const auto &p : map1) {
        cout << p.first << " -> " << p.second << endl;
    }

    for (const auto &p : map2) {
        cout << p.first << " -> " << p.second << endl;
    }

    return 0;
}
```

---

### 2. **通过初始化列表赋值**

C++11 起支持使用初始化列表直接对 `map` 初始化：

```cpp
#include <iostream>
#include <map>
using namespace std;

int main() {
    // 使用初始化列表赋值
    map<int, string> map1 = {{1, "One"}, {2, "Two"}, {3, "Three"}};
    map<string, int> map2 = {{"Alice", 30}, {"Bob", 25}, {"Charlie", 35}};

    // 输出验证
    for (const auto &p : map1) {
        cout << p.first << " -> " << p.second << endl;
    }

    for (const auto &p : map2) {
        cout << p.first << " -> " << p.second << endl;
    }

    return 0;
}
```

---

### 3. **使用构造函数初始化**

在 `std::map` 构造时直接传递键值对的初始化列表：

```cpp
#include <iostream>
#include <map>
using namespace std;

int main() {
    map<int, string> map1({{1, "One"}, {2, "Two"}, {3, "Three"}});
    map<string, int> map2({{"Alice", 30}, {"Bob", 25}, {"Charlie", 35}});

    // 输出验证
    for (const auto &p : map1) {
        cout << p.first << " -> " << p.second << endl;
    }

    for (const auto &p : map2) {
        cout << p.first << " -> " << p.second << endl;
    }

    return 0;
}
```

---

### 4. **通过 `emplace` 方法赋值**

`emplace` 是 C++11 引入的方法，避免了不必要的拷贝。

```cpp
#include <iostream>
#include <map>
using namespace std;

int main() {
    map<int, string> map1;
    map<string, int> map2;

    // 使用 emplace 添加元素
    map1.emplace(1, "One");
    map1.emplace(2, "Two");
    map2.emplace("Alice", 30);
    map2.emplace("Bob", 25);

    // 输出验证
    for (const auto &p : map1) {
        cout << p.first << " -> " << p.second << endl;
    }

    for (const auto &p : map2) {
        cout << p.first << " -> " << p.second << endl;
    }

    return 0;
}
```

---

### 5. **多 `map` 变量同时赋初值**

如果有多个 `map` 变量，可以通过函数或批量赋值的方式进行初始化：

```cpp
#include <iostream>
#include <map>
using namespace std;

int main() {
    // 多个 map 变量
    map<int, string> map1, map2;
    map<string, int> map3, map4;

    // 批量赋值
    map1 = {{1, "One"}, {2, "Two"}, {3, "Three"}};
    map2 = {{4, "Four"}, {5, "Five"}};
    map3 = {{"Alice", 30}, {"Bob", 25}};
    map4 = {{"Charlie", 35}, {"Dave", 40}};

    // 输出验证
    for (const auto &p : map1) cout << p.first << " -> " << p.second << endl;
    for (const auto &p : map2) cout << p.first << " -> " << p.second << endl;
    for (const auto &p : map3) cout << p.first << " -> " << p.second << endl;
    for (const auto &p : map4) cout << p.first << " -> " << p.second << endl;

    return 0;
}
```

---

### 总结

手动给多个 `map` 变量赋初值的方式有：

1. 直接插入（`insert` 或 `operator[]`）。
2. 初始化列表（`{{key, value}, ...}`）。
3. 构造函数初始化。
4. 使用 `emplace` 插入。
5. 批量赋值或封装函数处理。

选择合适的方法可以简化代码，提高可读性和效率。
