

### 问题 R：顺序数字串查询（Increasing Number String Query）

#### 题目描述

考虑一个由所有正整数按递增顺序拼接而成的无限长字符串：  
`12345678910111213141516171819202122232425...`

你的任务是处理 **q** 个查询，每个查询的形式是：在这个字符串中，位置 **k** 上的数字是什么？  
注意：位置是从 **1** 开始计数的。

#### 输入

第一行输入一个整数 **q**（1 ≤ q ≤ 1000）：表示查询的数量。  
接下来有 **q** 行，每行一个整数 **k**（1 ≤ k ≤ 10¹⁸）：表示要查询的字符串中的位置。

#### 输出

对于每个查询，输出对应的数字。

#### 样例输入
```
3
7
19
12
```

#### 样例输出
```
7
4
1
```
