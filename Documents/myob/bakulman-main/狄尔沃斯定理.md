狄尔沃斯定理在序列问题上的一个推论是：

**对于任意一个序列，将其划分为最少数量的“非递增子序列”的个数，恰好等于这个序列的“最长递增子序列 (LIS)” 的长度。**

我们来拆解一下关键点：

- **最长递增子序列 (Longest Increasing Subsequence, LIS)**: 这个你很熟悉，就是找一个子序列，里面的数字严格递增，并且长度最长。
- **非递增子序列 (Non-increasing Subsequence)**: 这意味着子序列中的数字是 `a >= b >= c ...`，允许相等。这与“严格下降” (`a > b > c ...`) 有细微差别。
- **划分 (Partition)**: 这意味着序列中的**每一个**元素都必须属于且仅属于其中一个“非递增子序列”。
