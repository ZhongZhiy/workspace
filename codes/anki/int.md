# 积分
## $\displaystyle \int x^n dx = ?$ （$n\ne -1$）

$\displaystyle \frac{x^{n+1}}{n+1} + C$

## $\displaystyle \int \frac{1}{x},dx = ?$

$\ln|x| + C$

## $\displaystyle \int \sqrt{x},dx = ?$

$\displaystyle \frac{2}{3}x^{3/2} + C$

---


## $\displaystyle \int e^x,dx = ?$

$e^x + C$

## $\displaystyle \int a^x,dx = ?$

$\displaystyle \frac{a^x}{\ln a} + C$

## $\displaystyle \int \ln x,dx = ?$

$x\ln x - x + C$



## $\displaystyle \int \sin x,dx = ?$

$-\cos x + C$

## $\displaystyle \int \cos x,dx = ?$

$\sin x + C$

## $\displaystyle \int \tan x,dx = ?$

$-\ln|\cos x| + C$

## $\displaystyle \int \cot x,dx = ?$

$\ln|\sin x| + C$

## $\displaystyle \int \sec^2 x,dx = ?$

$\tan x + C$

## $\displaystyle \int \csc^2 x,dx = ?$

$-\cot x + C$

---


## $\displaystyle \int \frac{1}{\sqrt{1-x^2}},dx = ?$

$\arcsin x + C$

## $\displaystyle \int \frac{1}{1+x^2},dx = ?$

$\arctan x + C$

## $\displaystyle \int \frac{1}{x\sqrt{x^2-1}},dx = ?$

$\arcsec x + C$

---


## $\displaystyle \int \sinh x,dx = ?$

$\cosh x + C$

## $\displaystyle \int \cosh x,dx = ?$

$\sinh x + C$

## $\displaystyle \int \tanh x,dx = ?$

$\ln(\cosh x) + C$

---


## $\displaystyle \int_{-\infty}^{\infty} e^{-x^2},dx = ?$

$\sqrt{\pi}$

## $\displaystyle \int_{-\infty}^{\infty} e^{-x^2/2},dx = ?$

$\sqrt{2\pi}$

## $\displaystyle \int_0^{\infty} x e^{-x^2},dx = ?$

$\frac{1}{2}$

## $\displaystyle \int_{0}^{\infty} e^{-a x^2},dx = ? \quad (a>0)$

$\displaystyle \frac{1}{2}\sqrt{\frac{\pi}{a}}$

## 连续型随机变量的期望公式：

<!-- notecardId: 1762673074365 -->

$\displaystyle E[X] = \int_{-\infty}^{\infty} x f(x),dx$

## 方差的积分表达式：

<!-- notecardId: 1762673074374 -->

$\displaystyle Var(X) = \int (x - E[X])^2 f(x),dx$
