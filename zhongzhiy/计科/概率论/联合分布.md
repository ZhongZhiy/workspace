# §2.4 随机向量与分布

## 1) 联合分布

**定义**（二维）：

- 分布函数 (F(x,y)=P(X\le x,,Y\le y))（见图示直观理解，图2-6，P5）。
    
- 连续型：若存在非负函数 (f(x,y)) 使得 (F(x,y)=\int_{-\infty}^x\int_{-\infty}^y f(u,v),dv,du)，称 (f) 为联合密度（P11–12）。
    
- 离散型：(p_{ij}=P{X=x_i,Y=y_j})；表格展示（P7–8）。
    

**性质（记忆框）**（P6）：

- (0\le F\le 1)，单调，右连续；(\lim_{x\to-\infty}F(x,y)=0)、(\lim_{x,y\to+\infty}F=1)；矩形不等式非负。
    

**常见分布**

- 平面区域 (G) 上均匀：(f(x,y)=1/A(G)) 于 (G)，其余0（P15）。
    
- 二维正态 (N(\mu_1,\mu_2;\sigma_1,\sigma_2;\rho))（公式与三维图见P16–19）。其边缘都是正态（P31）。
    

**例**

- 例2-11：离散型构表 + 乘法公式求 (p_{ij})（P9–10）。
    

## 2) 边缘分布

**离散**：(p_{i\cdot}=\sum_j p_{ij})、(p_{\cdot j}=\sum_i p_{ij})（P22–24, 36）。  
**连续**：(f_X(x)=\int_{-\infty}^{+\infty}f(x,y),dy)，(f_Y(y)=\int f(x,y),dx)（P25–26）。  
**例**

- 例2-14：区域 (y=x^2) 与 (y=x) 间均匀分布，先算面积再得联合密度→再积分得边缘（P26–27）。
    
- 例2-15：二维正态的边缘密度仍为正态（P28–31）。
    

## 3) 条件分布

**离散**：(P{X=x_i|Y=y_j}=p_{ij}/p_{\cdot j})（P36–37）。  
**连续**：(f_{X|Y}(x|y)=\dfrac{f(x,y)}{f_Y(y)})，(f_{Y|X}(y|x)=\dfrac{f(x,y)}{f_X(x)})；且  
(f(x,y)=f_Y(y)f_{X|Y}(x|y)=f_X(x)f_{Y|X}(y|x))（P38–41）。  
**几何例**：圆盘 (x^2+y^2\le1) 上均匀→(f_{X|Y}(x|y)=\dfrac{1}{2\sqrt{1-y^2}})（(|x|\le\sqrt{1-y^2})）(P42)。

**解题套路**

1. 先定联合（表/密度）→ 2) 求边缘 → 3) 条件（用比值）→ 4) 再判断独立/计算概率（区域积分要画图）。

# §2.5 独立性

**判定**

- 分布函数判定：(F(x,y)=F_X(x)F_Y(y))（P44–45）。
    
- 连续型等价于密度因子分解：(f(x,y)=f_X(x)f_Y(y))（P45–46）。
    
- 二维正态：独立 **充要** 条件 (\rho=0)（P47, 90）。
    

**反例提醒**

- 仅靠单个格点 (p_{11}\stackrel{?}{=}p_{1\cdot}p_{\cdot1}) 不等于独立；需对**所有**取值成立（P49–51 对照）。
    

**例**

- 例2-17：已知边缘都正态且“独立”→联合密度为两者乘积（P46–47）。
    
- 例2-18/2-19：用边缘乘积与联合对比判否独立（P48–49）。
    
- 配套练习表格反推 (p_{ij})（P50–51）。

# §2.6 随机变量函数的分布

## 1) 单变量变换 (Y=g(X))

**离散**：把相同像值合并，概率相加（P56–59）。  
**连续（单调可导）**：**换元公式**（定理2-4，P62,68）  
[  
f_Y(y)=f_X!\big(h(y)\big)\cdot \big|h'(y)\big|,\quad h=g^{-1}.  
]  
**非单调**：用“分布函数法” (F_Y(y)=P(g(X)\le y))（P65–66）。  
**例**

- 线性变换 (Y=aX+b\Rightarrow Y\sim N(a\mu+b,a^2\sigma^2))（P63–64, 88(3)）。
    
- (Y=X^2)（标准正态）：分段求导得密度（P65–66）。
    
- 练习：指数/对数/反正切等（P69–70, 77）。
    

**易错点**：别忘 **(|h'(y)|)**；非单调必须用 CDF 法分段。

## 2) 两变量函数：和、极值

**和 (Z=X+Y)**（一般）：  
[  
f_Z(z)=\int_{-\infty}^{+\infty} f(x,z-x),dx=\int f(z-y,y),dy\quad(\text{P72}).  
]  
**独立时（卷积）**：(f_Z=f_X*f_Y)（P73–75）。

- 结论：独立正态之和仍正态，(\mu) 相加、方差相加（P76, 89）。
    

**极大/极小（独立）**（P78–79）：  
[  
F_{\max}(z)=F_X(z)F_Y(z),\quad  
F_{\min}(z)=1-\big(1-F_X(z)\big)\big(1-F_Y(z)\big).  
]  
可推至 n 个独立变量：(F_{\max}=F^n,;F_{\min}=1-(1-F)^n)（P79）。

**例**

- 例2-23：两独立 (N(0,1)) 卷积得 (Z\sim N(0,2))（P74–76）。
    
- 例2-25：(\max,\min) 的分布函数（P78）。
    
- 例2-27：(f(x,y)=A e^{-(x+2y)}, x>0,y>0)。由归一化得 (A=2)，边缘 (f_X=e^{-x}\mathbf{1}_{x>0}, f_Y=2e^{-2y}\mathbf{1}_{y>0})，故独立；再求 (Z=X+2Y) 的分布（P85–87）。
    
- 练习：(X\sim U(0,1), Y\sim \mathrm{Exp}(1)) 的和（分段卷积，P98–99 图示区域）。
    

**应用**

- **系统可靠性**：串联→(\min)，并联→(\max)，备用→和（卷积）。独立指数族下：
    
    - 串联：(\min\sim \mathrm{Exp}(\alpha+\beta))（P81）。
        
    - 并联：(F_{\max}(z)=(1-e^{-\alpha z})(1-e^{-\beta z}))（P82）。
        
    - 备用：密度分段式 (f_Z(z)=\dfrac{\alpha\beta}{\beta-\alpha}\big(e^{-\alpha z}-e^{-\beta z}\big))（P83）。
        
- **灯泡模型**（P100–103）：增加灯泡数使“首次失效时间”从 (\min{4}) 变为 (\min{24})，独立指数 → 失效更“密”（更早），因此“更换更频繁”的直觉**有依据**（用 (P(T\ge t)=e^{-k\lambda t}) 比较）。

# 公式快表（A4速览）

- **联合→边缘**：  
    (\displaystyle f_X(x)=\int f(x,y),dy,\quad f_Y(y)=\int f(x,y),dx).  
    (\displaystyle p_{i\cdot}=\sum_j p_{ij},;p_{\cdot j}=\sum_i p_{ij}).
    
- **条件密度/分布**：  
    (\displaystyle f_{X|Y}(x|y)=\frac{f(x,y)}{f_Y(y)}),  
    (f(x,y)=f_Y(y)f_{X|Y}(x|y)=f_X(x)f_{Y|X}(y|x)).
    
- **独立判定**：(F=F_XF_Y)；连续型等价 (f=f_Xf_Y)；二维正态独立⇔(\rho=0)。
    
- **单调变换**：(f_Y(y)=f_X(h(y))|h'(y)|)。
    
- **卷积**：(\displaystyle f_Z(z)=\int f_X(x)f_Y(z-x),dx)（独立）。
    
- **极值（独立）**：(F_{\max}=F_XF_Y)，(F_{\min}=1-(1-F_X)(1-F_Y))。

# 解题通用套路（拿来就用）

1. **画图**：先画支持域/事件区域（如图2-6、P71、P98）。
    
2. **先边缘后条件**：从 (f) 或 (p_{ij}) 出发 → 边缘 → 条件/独立性。
    
3. **变换优先选换元**：单调可导用换元；否则用 CDF 法分段。
    
4. **卷积分段**：先找被积函数非零区间，再分段积分。
    
5. **正态加和**：直接“均值加、方差加”。
    
6. **指数最小值**：参数相加仍指数；备用为差指数（看 P81–83）。

- 把“(\rho=0)”当成“一定独立”（**仅**二维正态成立，P47, 90）。
    
- 单变量变换忘了 (|h'(y)|)；非单调却用换元。
    
- 卷积不分段直接积分；未先确定非零区。
