| 模式简称 | 模式名称                    | 说明喵~                                   |
| ---- | ----------------------- | -------------------------------------- |
| `n`  | Normal 普通模式             | 默认模式，可以移动、复制、粘贴等                       |
| `i`  | Insert 插入模式             | 输入文本用的，按 `i`、`a`、`o` 等就能进这个模式          |
| `v`  | Visual 可视模式             | 选中文本的模式，按 `v` 进入                       |
| `x`  | Visual Block 块可视模式      | 用 `Ctrl-v` 进入，选中的是“矩形区域”               |
| `s`  | Select 选择模式             | 类似 Visual 模式，常在图形界面用（不常用），选中一部分文字，进行替换 |
| `o`  | Operator-pending 操作等待模式 | 例如你按了 `d`，但还没告诉它删哪，这时候就进入了 o 模式        |
| `t`  | Terminal 终端模式           | 如果你用 `:terminal` 打开了终端，就会进这个模式         |
| `c`  | Command-line 命令行模式      | 输入 `:`、`/`、`?` 时的模式                    |
| `l`  | Lang-Arg 输入法相关模式        | 极少使用，和多语言输入法切换相关                       |


不同模式对应不同的[[vim键盘映射]]
