### ✅ shutdown 指令详解

#### 💡 问答卡

```
## shutdown 指令常用参数有哪些？
-k ：仅发出警告，不实际关机  
-r ：关闭服务后重启系统  
-h ：关闭服务后关机  
-c ：取消已排定的 shutdown 行为
```

```
## 如何使用 shutdown 指令关闭系统并显示提示？
格式：`shutdown [选项] [时间] [警告讯息]`
```

```
## shutdown 指令中，若不指定时间会发生什么？
系统默认 1 分钟后执行关机
```

#### 💡 填空卡

```
## shutdown 指令中，{{c1::-r}} 是重启系统，{{c2::-h}} 是关机。
```

```
## shutdown 中的 {{c1::-k}} 参数只会发出警告，不会实际关机。
```

### ✅ man page 结构与分类

#### 💡 问答卡

```
## man page 的常见章节有哪些？简要说明其作用。
- NAME：简要描述命令
- SYNOPSIS：语法格式
- DESCRIPTION：详细说明
- OPTIONS：可用选项说明
- COMMANDS：可用子命令
- FILES：使用到的文件

- EXAMPLE：参考使用范例
```

```
## man page 后面的数字分别代表什么意思？例如 man(1)、man(5)？
1：用户指令  
2：系统调用  
3：C 语言库函数  
4：设备文件说明  
5：配置文件格式  
6：游戏  
7：惯例、协议说明  
8：系统管理指令  
9：内核相关文件
```

```
## 如何查找所有含有关键字 "man" 的 man page？
使用 `man -k man` 或 `apropos man`
```

**## Linux 系統中的檔案名稱是否區分大小寫？**

**## 使用 `man date` 時發現亂碼可能是什麼原因？怎麼修復？**  
可能是語系設定錯誤，修復方式為：`export LANG=en_US.utf8` 或 `export LC_ALL=en_US.utf8`

---

**## 輸入 `ls -al /vbird` 回傳 No such file or directory，代表什麼？**

**## 如何查出系統中以 `bz` 開頭的指令有哪些？**  
在終端輸入 `bz[Tab][Tab]` 利用補齊功能即可列出所有以 `bz` 開頭的指令。

---

**## `bzip2` 指令的作用是什麼？**  
它是用來對檔案進行壓縮與解壓縮的工具。

---

**## `#` 與 `$` 提示字元的差別是什麼？應該用哪一個操作系統？**  
`#` 表示 root 使用者，`$` 表示一般使用者。日常操作建議使用一般使用者（`$`）。

---

**## 一般帳號可以直接用 `reboot` 重啟系統嗎？**  
不行，`reboot` 需要 root 權限；除非圖形介面提供 GUI 重啟功能。
