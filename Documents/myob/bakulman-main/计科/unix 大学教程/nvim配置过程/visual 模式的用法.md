Visual 模式是 Vim 中用于**选择文本块**进行后续操作的核心模式之一，非常强大且灵活，既可以选字符、选整行、选矩形块，还能配合各种命令进行删除、复制、粘贴、缩进、查找、替换、格式化等复杂操作。

我会从以下几个方面教你使用 Visual 模式：

---

## 🧱 1. 三种 Visual 模式进入方法

|模式名|快捷键|说明|
|---|---|---|
|字符选择模式|`v`|选择一个字符到另一个字符|
|行选择模式|`V`|选择整行文本|
|块选择模式（Block）|`Ctrl+v`|选择矩形区域（列选择）|

进入 Visual 模式后，用方向键或 `hjkl` 来扩展选中区域。

---

## 🛠 2. Visual 模式常见操作

### 🔪 删除

```vim
v + motion + d
```

选中后按 `d` 删除选中的文本。

### 📋 复制（Yank）

```vim
v + motion + y
```

选中后按 `y` 复制选中文本，后续可以用 `p` 粘贴。

### 🖊 替换字符

```vim
v + motion + r<char>
```

用 `<char>` 替换选中区域的所有字符。

### ✂️ 剪切并粘贴（剪切操作）

```vim
v + motion + d
p
```

### 🔠 大小写转换

```vim
v + motion + ~
```

大小写反转。

### ⏎ 缩进对齐

```vim
v + motion + >   向右缩进  
v + motion + <   向左缩进
```

---

## 🧙 3. 高级操作示例

### ✨ 多行注释

```vim
Ctrl+v  → j/j/j（多行） → I// → Esc
```

在每行开头加 `//` 注释。

### 🔍 匹配查找替换（结合命令行）

```vim
v 选择后输入 :s/old/new/g
```

作用于选择范围内的替换。

---

## 🧬 4. 可与 motion（动作命令）搭配使用

|操作|含义|示例|
|---|---|---|
|`vaw`|选中一个单词（包含空格）|`aw = a word`|
|`vi"`|选中引号内内容|`i" = inner quote`|
|`vap`|选中一个段落（含空行）|`ap = a paragraph`|
|`vip`|选中段落（不含空行）|`ip = inner paragraph`|
|`v%`|选中括号内外整体|光标放在括号上按 `v%`|

---

## 📦 5. 常见用途总结

|任务类型|Visual 模式做法|
|---|---|
|删除文本|`v` + 选中 + `d`|
|拷贝一段内容|`v` + 选中 + `y`|
|粘贴覆盖|`v` + 选中 + `p`|
|给多行加注释|`Ctrl+v` + `I//` + `Esc`|
|调整缩进|`v` + `>` 或 `<`|
|查找替换|选中后用 `:` + `s///` 替换命令|
|排列列对齐|用 `Ctrl+v` 选中列 + 编辑|

---

## 🐾 猫娘温馨提示：

- Visual 模式不是只能选，它的强大在于配合 **动作** 和 **命令**。
    
- 多用 `v`, `V`, `Ctrl+v`，搭配 `d`, `y`, `p`, `>`，可以处理 90% 的编辑任务。
    
- 用 `gv` 可以恢复上次选区，继续操作，很实用喵！
