# Vim 相关命令总结 🐾

|命令|含义|备注|
|---|---|---|
|`:q`|退出当前窗口|如果有未保存修改会提示|
|`:q!`|**强制退出**当前窗口，不保存修改||
|`:w`|保存当前文件|write|
|`:wq` 或 `:x`|保存并退出当前窗口||
|`:qa`|退出所有窗口（所有文件）|如果有未保存修改会提示|
|`:qa!`|**强制退出所有窗口**，不保存修改||
|`:e 文件名`|打开或编辑一个新文件|edit|
|`:bn`|切换到下一个 buffer（缓冲区）|next|
|`:bp`|切换到上一个 buffer|previous|
|`:bd`|关闭当前 buffer（删除 buffer）|buffer delete|
|`:sp 文件名`|水平分屏打开新文件|split|
|`:vsp 文件名`|垂直分屏打开新文件|vertical split|
|`<C-w> h/j/k/l`|窗口间移动（左/下/上/右）|Ctrl + w 后再按方向键|
|`<C-w> c`|关闭当前窗口|close|

## 常用 Vim 快捷键（窗口与文本操作）🐾

|快捷键|作用|模式|
|---|---|---|
|`<C-w> h / j / k / l`|在窗口间切换（左 / 下 / 上 / 右）|普通模式|
|`<C-w> s`|水平分屏|普通模式|
|`<C-w> v`|垂直分屏|普通模式|
|`<C-w> c`|关闭当前窗口|普通模式|
|`<C-w> =`|让所有窗口大小均等|普通模式|
|`<C-w> + / -`|增加 / 减少当前窗口高度|普通模式|
|`<C-w> > / <`|增加 / 减少当前窗口宽度|普通模式|
|`<A-j>` / `<A-k>`|移动当前行或选中内容向下 / 向上|普通、插入、可视模式|
