在 C++ 中，可以使用构造函数对结构体的成员变量进行初始化。C++11 及以上版本还支持直接在结构体声明中定义构造函数和使用初始化列表。这种方法既方便又安全。

以下是一些常见的方式：

---

### **1. 定义构造函数**

在结构体中定义一个构造函数，通过构造函数初始化成员变量。

**代码示例：**

```cpp
#include <iostream>
using namespace std;

// 定义结构体
struct MyStruct {
    int a;
    double b;

    // 构造函数
    MyStruct(int x, double y) {
        a = x;  // 初始化成员变量 a
        b = y;  // 初始化成员变量 b
    }

    // 成员函数，打印结构体数据
    void print() {
        cout << "a = " << a << ", b = " << b << endl;
    }
};

int main() {
    MyStruct obj(10, 3.14);  // 调用构造函数初始化
    obj.print();             // 输出: a = 10, b = 3.14

    return 0;
}
```

---

### **2. 使用初始化列表**

推荐使用初始化列表来初始化结构体的成员变量，语法简洁且效率更高（直接初始化而不是先默认构造再赋值）。

**代码示例：**

```cpp
#include <iostream>
using namespace std;

struct MyStruct {
    int a;
    double b;

    // 使用初始化列表
    MyStruct(int x, double y) : a(x), b(y) {}

    void print() {
        cout << "a = " << a << ", b = " << b << endl;
    }
};

int main() {
    MyStruct obj(20, 6.28);  // 调用构造函数并初始化
    obj.print();             // 输出: a = 20, b = 6.28

    return 0;
}
```

---

### **3. 默认构造函数**

如果结构体不带参数的构造函数，可以为成员变量提供默认值。

**代码示例：**

```cpp
#include <iostream>
using namespace std;

struct MyStruct {
    int a = 0;     // 直接在定义时初始化
    double b = 0;  // C++11 支持直接初始化成员变量

    MyStruct() {}  // 默认构造函数（无需初始化，因为已有默认值）

    void print() {
        cout << "a = " << a << ", b = " << b << endl;
    }
};

int main() {
    MyStruct obj;  // 调用默认构造函数
    obj.print();   // 输出: a = 0, b = 0

    return 0;
}
```

---

### **4. 提供多个构造函数（构造函数重载）**

可以定义多个构造函数，支持不同的初始化方式。

**代码示例：**

```cpp
#include <iostream>
using namespace std;

struct MyStruct {
    int a;
    double b;

    // 默认构造函数
    MyStruct() : a(0), b(0) {}

    // 带参数的构造函数
    MyStruct(int x, double y) : a(x), b(y) {}

    void print() {
        cout << "a = " << a << ", b = " << b << endl;
    }
};

int main() {
    MyStruct obj1;         // 调用默认构造函数
    MyStruct obj2(42, 1.23); // 调用带参数的构造函数

    obj1.print();          // 输出: a = 0, b = 0
    obj2.print();          // 输出: a = 42, b = 1.23

    return 0;
}
```

---

### **5. 使用 `std::initializer_list`（C++11 新特性）**

可以通过 `std::initializer_list` 来支持结构体的统一初始化。

**代码示例：**

```cpp
#include <iostream>
#include <initializer_list>
using namespace std;

struct MyStruct {
    int a;
    double b;

    // 使用 initializer_list
    MyStruct(initializer_list<int> init) {
        auto it = init.begin();
        a = *it;
        b = *(it + 1);
    }

    void print() {
        cout << "a = " << a << ", b = " << b << endl;
    }
};

int main() {
    MyStruct obj{10, 20};  // 使用大括号初始化
    obj.print();           // 输出: a = 10, b = 20

    return 0;
}
```

---

### **6. 简单结构体的聚合初始化**

如果结构体只包含 `public` 成员且没有用户定义的构造函数，可以直接用聚合初始化方式赋值。

**代码示例：**

```cpp
#include <iostream>
using namespace std;

struct MyStruct {
    int a;
    double b;
};

int main() {
    MyStruct obj = {10, 3.14};  // 聚合初始化
    cout << "a = " << obj.a << ", b = " << obj.b << endl;

    return 0;
}
```

---

### **总结**

- **直接赋值**：适合简单结构体，用 `{}` 聚合初始化。
- **初始化列表**：推荐，效率高、语法简洁。
- **构造函数重载**：适合需要支持多种初始化方式的场景。
- **默认值 + 默认构造函数**：适合需要默认值的成员变量。
- **`std::initializer_list`**：适合初始化多个成员变量时提供统一接口。

选择哪种方法取决于你的需求和代码的复杂性。
