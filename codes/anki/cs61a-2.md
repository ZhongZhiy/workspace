## 什么是高阶函数（Higher-Order Function）？

高阶函数是指**将函数作为参数或返回值的函数**。

---

## 高阶函数的作用是什么？

它是一种抽象：**用函数表达通用计算模式，而与被调用的具体函数无关**。

---

## 为什么高阶函数比普通函数更强大？

因为它能**抽象计算过程**，不仅仅执行具体操作，还能把函数本身当作计算单元传递。

---

## 控制表达式（control expression）与函数调用的区别是什么？

**控制表达式**（如 if）可以**跳过某些分支**；
而函数调用会**计算所有参数后再执行函数体**，不会跳过任何参数。

---

## Python 中 if 语句的求值特性是什么？

`if` 语句**按条件分支执行**，只有条件为真时才求值对应分支。

---

## Python 中逻辑运算符 `and` 与 `or` 的短路行为是什么？

* `and` 在左值为假时不再求右值。
* `or` 在左值为真时不再求右值。

---

## 举例：用函数实现 if 表达式会有什么问题？

若写成 `if_fn(cond, true_expr, false_expr)`，则 `true_expr` 与 `false_expr` 都会先被求值；
这与真正的 if 不同，**无法跳过无用分支**。

---

## `assert` 语句的作用是什么？

断言某条件为真；若条件为假，则抛出异常并输出提示信息。

---

## `make_adder` 函数的定义是什么？

```python
def make_adder(n):
    def adder(k):
        return k + n
    return adder
```

---

## 调用 `make_adder(1)(2)` 的结果是什么？

返回 `3`。

---

## 为什么 `make_adder` 能访问外层变量 `n`？

因为 Python 具有**词法作用域（lexical scoping）**；
内部函数能**携带定义时的外部变量环境**，即使外层函数已结束执行。

---

## 什么是 first-class object？

指函数可以像值一样：

* 赋给变量；
* 作为参数；
* 作为返回值；
* 存储在数据结构中。

---

## 什么是 currying（柯里化）？

将一个接受多个参数的函数，转换为**一系列每次只接受一个参数的函数**。

---

## 举例：柯里化（currying）的形式？

若原函数为 `f(x, y)`，柯里化后可写作：

```python
f_curried = lambda x: lambda y: f(x, y)
```

---

## 什么是 lambda 表达式？

一种**匿名函数表达式**，语法为：

```python
lambda 参数: 返回表达式
```

---

## lambda 表达式中不能使用什么？

不能包含赋值或控制语句，只能有一个返回表达式。

---

## 举例：lambda 表达式的含义？

```python
lambda x: f(g(x))
```

表示“一个接受 `x` 并返回 `f(g(x))` 的函数”。

---

## 什么是函数装饰器（decorator）？

一种**在函数定义时自动修改或包装函数行为**的语法糖。

---

## 装饰器的基本语法？

```python
@函数名
def 原函数(...):
    ...
```

---

## 装饰器等价于哪种赋值形式？

```python
原函数 = 函数名(原函数)
```

---

## 为什么装饰器是高阶函数？

因为它**接受一个函数作为参数，并返回一个新函数**。

---

## 高阶函数与闭包的关系？

高阶函数可以生成闭包；闭包保存了外层函数的变量环境，使内部函数能在外层结束后仍访问外部变量。

---

## Python 中局部函数（inner function）的作用？

定义在函数内部的函数，可用来封装逻辑并形成闭包，**增强抽象与复用性**。

---

## 为什么高阶函数能实现“计算通用化”？

因为它把**操作与数据分离**，可通过传入不同函数来复用相同计算结构。

---

## 举例：高阶函数控制计算流程的典型应用？

如：

```python
map(func, iterable)
filter(predicate, iterable)
reduce(func, iterable)
```
