1. **通过开始菜单或搜索栏:**
    
    - 点击 Windows 开始按钮或按 `Win` 键。
    - 输入 "PowerShell"。
    - 在搜索结果中找到 "Windows PowerShell"。
    - **右键单击** "Windows PowerShell"。
    - 选择 "**以管理员身份运行**" (Run as administrator)。
    - 如果出现用户账户控制 (UAC) 提示，请点击 "是" (Yes)。
2. **通过 "运行" 对话框:**
    
    - 按下 `Win + R` 键打开 "运行" 对话框。
    - 输入 `powershell`。
    - **同时按下 `Ctrl + Shift + Enter`** 键。
    - 如果出现用户账户控制 (UAC) 提示，请点击 "是" (Yes)。
3. **通过任务管理器:**
    
    - 按下 `Ctrl + Shift + Esc` 打开任务管理器。
    - 点击菜单栏的 "文件" (File)。
    - 选择 "运行新任务" (Run new task)。
    - 输入 `powershell`。
    - **勾选** "**创建此任务时使用管理员权限**" (Create this task with administrative privileges) 复选框。
    - 点击 "确定" (OK)。
4. **通过 Windows Terminal (如果已安装):**
    
    - 打开 Windows Terminal。
    - 点击窗口顶部标签栏旁边的**向下箭头 (v)**。
    - **按住 `Ctrl + Shift` 键**，然后点击列表中的 "Windows PowerShell" 配置文件。
    - 或者，右键单击任务栏或开始菜单中的 Windows Terminal 图标，然后选择 "以管理员身份运行"。
