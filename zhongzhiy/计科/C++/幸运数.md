### 问题 D: 幸运数III

时间限制: 1.000 Sec  内存限制: 128 MB  

#### 题目描述

4 和 7 是味味的幸运数字。幸运数是那些只由幸运数字组成的正整数。如 47，477 是幸运数，而 5，17，417 就不是幸运数。  
定义 next(x)为大于或等于 x 的最小的幸运数。  
味味对以下表达式的值很感兴趣 ：  
next(L)+next(L+1)+...+next(R-1)+next(R)。  
现在告诉你 L 和 R 的值，希望你能帮助味味计算出这个表达式的值。  

#### 输入

输入仅一行包含两个正整数 L 和 R(1≤L≤R≤109 )，L 和 R 的值之间用一个空格分隔。

#### 输出

输出只有一行一个整数，表示表达式的值。

#### 样例输入 Copy

2 7

#### 样例输出 Copy

33

#### 提示

next(2)+next(3)+next(4)+next(5)+next(6)+next(7)=4+4+4+7+7+7=33  
  
对于 20%的数据，1≤L≤R≤1000  
对于 40%的数据，1≤L≤R≤106  
另有 20%的数据，L=R  
对于 100%的数据，1≤L≤R≤109

#include<bits/stdc++.h>

vector<ll> v;
void dfs(int d, ll n) {
	if(d == 0) {
		v.emplace_back(n);
		return;
	}

	dfs(d-1, n*10 + 4);
	dfs(d-1, n*10 + 7);
}
int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	cout.tie(0);

	for(int i = 1;i <= 9; ++i) {
		dfs(i, 4);
		dfs(i, 7);
	}

	v.emplace_back(4);
	v.emplace_back(7);

	sort(v.begin(), v.end());

	ll l, r;
	cin  >> l >> r;

	ll b = 0, f = 0, ans = 0;
	auto iu = lower_bound(v.begin(), v.end(), l) - v.begin();
	f = v[iu];
	for(int i = l;i <= r; ++i) {
		if(i-1 != f) ans += f; 
		else {
			iu++;
			ans += v[iu];
			f = v[iu];
		}
	}

	cout << ans << endl;
	return 0;

}
