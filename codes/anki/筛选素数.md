## 欧拉筛第一步

<!-- notecardId: 1759841972687 -->
收集素数, 如果这个数是素数, 就收集
```cpp
if(!not_pri[i])
  primer.push_back(i);
```

## 欧拉筛第二步

<!-- notecardId: 1759841972697 -->
便利所有素数, 用当前数`i`乘所有素数p, 打标记, 代表这些数的最小素数就是p, 当前`i`是素数p的倍数的时候, 停止
```cpp
for(auto p : primer){
  if(p * i > maxn) break;
  not_pri[i * p] = 1;
  if(i % p == 0) break;
}
```

## 埃氏筛

<!-- notecardId: 1759841972702 -->
```cpp
for(int i = 2;i * i<= N; ++i) // sqrt(n)范围就可以了
{
  if(!not_pri[i]){
    for(int j = i * i; j <= N; j += i) {
      not_pri[j] = 1;
    }
  }
}
```
